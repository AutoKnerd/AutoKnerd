# AutoKnerd Web

This is the front-end repository for the AutoKnerd website. Built using Next.js and Tailwind CSS with a premium technology aesthetic.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
